    Enrollment System

    Project Details

    A simple enrollment system made using Handlebars, SASS, Node.js and utilizing MySQL for database.

  • Admin or student can both sign up for an account.
  • Upon login, student is redirected to his or her student profile.
  • Upon login, admin is redirected to admin page.
  • Admin can view list of students and list of admins.
  • Admin can add, edit, or delete student enrollment record.

    • Client: None
    • Date: August 2021
    • Code: github.com/jmcarriedo/nodejs-enrollment
